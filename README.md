# Lost-Bullet-Github
생존 서바이벌 크래프팅 게임을 2D Pixel로
재해석 한 게임입니다.
## 게임 소개
2D 인디 플랫포머 게임이며, 기존의 플랫포머 게임들은 단기간, 횡 스크롤 방식으로 매우 빠른 진행을 가지고 있지만, 
생존 서바이벌 등 여러가지 등을 병합하여 장시간 즐길 수 있는 게임입니다.
스토리를 진행하면서 다양한 임무와 무기, 파츠 조합으로 같은 임무여도 다양한 방식으로 플레이가 가능합니다.
## 주요 기능
- 다양한 임무 시스템
- 제작 시스템
- 생존 전략
